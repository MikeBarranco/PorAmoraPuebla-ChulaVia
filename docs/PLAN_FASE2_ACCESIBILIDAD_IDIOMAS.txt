EL PROBLEMA REAL QUE EL MENTOR SENALO
================================================================================

En las comunidades rurales de Puebla existe un problema que va mas alla de
"no hay transporte": la mayoria de soluciones digitales no llegan a quien
mas las necesita porque:

1. ANALFABETISMO: Muchas personas mayores en comunidades indigenas no saben
   leer ni escribir en espanol. El INEGI reporta que en municipios de alta
   marginacion el indice de analfabetismo supera el 30%.

2. LENGUA: El espanol es segundo idioma para muchos habitantes de la Sierra
   Norte, Totonacapan, y regiones Nahuatl. Hay 8 lenguas originarias en Puebla:
   Nahuatl, Totonaco, Mixteco, Mazateco, Popoloca, Otomi, Tepehua y Chocho.

3. UBICACION: "Se donde vivo pero no se como se llama en el mapa." Las
   comunidades se identifican por el nombre de la iglesia, el cerro, el rio,
   el santo patron. No por el nombre oficial del INEGI. Una abuela sabe que
   vive "junto a la iglesia de San Marcos" pero no sabe que eso se llama
   "San Marcos Tlacoyalco" en los registros oficiales.

4. TECNOLOGIA: Para una persona mayor, tocar una pantalla y escribir texto
   ya es una barrera. Necesitan flujos guiados por voz o por imagenes.

NUESTRA SOLUCION:
  - Voz: el usuario habla en vez de escribir
  - Mapa visual: toca tu comunidad en el mapa, sin escribir nada
  - Idiomas: la interfaz en espanol, nahuatl o totonaco
  - WhatsApp accesible: menus numerados + voz en el bot
  - Iconos grandes: para quien no puede leer, los iconos guian el flujo

BLOQUE 6 - MAPA: TIEMPO DE VIAJE + FOTO DEL CONDUCTOR
Tiempo estimado: 1 hora
Archivo: frontend/src/pages/MapPage.jsx

Tiempo de viaje con formula Haversine (distancia en km / 45 km/h promedio rural):
  function haversineKm(lat1, lon1, lat2, lon2) {
    const R = 6371
    const dLat = (lat2-lat1)*Math.PI/180
    const dLon = (lon2-lon1)*Math.PI/180
    const a = Math.sin(dLat/2)**2 +
      Math.cos(lat1*Math.PI/180)*Math.cos(lat2*Math.PI/180)*Math.sin(dLon/2)**2
    return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a))
  }
  function tiempoEst(origenNombre, destinoNombre) {
    const o = comunidades.find(c => c.nombre===origenNombre)
    const d = comunidades.find(c => c.nombre===destinoNombre)
    if (!o || !d) return '~30 min'
    const km = haversineKm(o.lat, o.lng, d.lat, d.lng)
    const min = Math.round(km/45*60)
    return min<60 ? `~${min} min` : `~${Math.floor(min/60)}h ${min%60}min`
  }
